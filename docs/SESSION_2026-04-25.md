## GA4 dashboard queries to run (1-7d after data starts flowing)

These give immediate decision-grade insight:

```
event_name = tool_view
GROUP BY tool_id
ORDER BY count DESC
→ Which tools get the most traffic?

event_name IN (tool_view, tool_upload, tool_run_success)
GROUP BY tool_id
→ Visit→upload→run funnel per tool. Drop-off at view→upload = bad
landing copy. Drop-off at upload→run = signup wall friction or
pricing-blurb anxiety.

event_name = tool_run_success
SUM(credit_cost) / SUM(events) GROUP BY tool_id
→ Average revenue per run, by tool. Identifies the high-ARPU
wedges to double down on.

event_name = signup_redirect
GROUP BY tool_id
→ Highest = best ad-spend target. Anonymous users went all the way
to 'Run' and bounced to login. They're hot leads.

event_name = tool_run_error
GROUP BY tool_id, error_code
→ Real bug surface. parse_failed = AI prompt issue.
http_402 = pricing problem. http_502 = provider degradation.

event_name = tool_run_success
AVG(processing_ms) GROUP BY depth
→ Latency p50 by AI prompt. Slow prompts hurt conversion.
```
